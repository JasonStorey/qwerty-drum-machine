# qwerty-drum-machine

Record, edit, and play audio samples straight from your browser (or, more accurately, Chrome).

<img src="http://jasonstorey.co.uk/demos/qwerty-drum-machine/qdm-screenshot.png" />

# Features

- Record Audio
- Edit in/out points
- Live sample triggering
- Low latency
